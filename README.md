# «Venevsky»
> Адаптивная вёрстка интернет магазина Venevsky по продаже строительных материалов.

## Технологии:
* pug / scss
* jquery
* yandex-maps
* [сборка gulp](https://github.com/uqrock/gulp-builder-scss-pug)

Поддержка: последние пять версий стабильных браузеров и ie11+.
